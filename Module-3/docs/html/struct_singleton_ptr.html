template<class T>
struct SingletonPtr< T >

Utility class for creating an using a singleton

@Note Synchronization level: Thread safe

@Note: This class must only be used in a static context - this class must never be allocated or created on the stack.

@Note: This class is lazily initialized on first use. This class is a POD type so if it is not used it will be garbage collected.
